# Stitch Frontend Export

Source: Google Stitch project `14435121892916306318` (`Car Rental Platform Frontend`)

Exported via MCP Stitch on 2026-05-12.

## Contents

- `screen-inventory.md`: route and role map exported from Stitch.
- `design-system.md`: local snapshot of the RentFlow design system returned by Stitch.
- `manifest.json`: machine-readable list of downloaded prototypes.
- `html/`: standalone HTML prototypes generated by Stitch.
- `screenshots/`: screenshots paired with the downloaded HTML prototypes.
- `download-stitch-assets.ps1`: repeatable downloader for the selected Stitch screen assets.

## Open A Prototype

Open any file under `html/` directly in a browser. The HTML files use external CDN assets such as Tailwind, Font Awesome, and Google Fonts, so the rendered pages need network access for exact styling.

Main entry points:

- `html/guest-landing-page.html`
- `html/guest-search-results.html`
- `html/guest-listing-detail.html`
- `html/customer-book-your-ride.html`
- `html/host-dashboard.html`
- `html/admin-system-overview.html`
- `html/global-component-kit.html`
