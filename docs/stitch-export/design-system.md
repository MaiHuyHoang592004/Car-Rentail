# RentFlow Design System Snapshot

Source: Google Stitch project `14435121892916306318`

## Brand

RentFlow is styled as a professional car rental marketplace with a minimalist corporate-modern interface. The design direction emphasizes trust, efficiency, task completion, and premium utility.

## Component Notes

- Car cards use high-quality media, a category label, a model title, visible specs, and clear pricing.
- Primary buttons use RentFlow blue with white text.
- Conversion actions such as booking use the orange accent.
- Inputs use a light outline, blue focus border, and subtle outer glow.
- Status badges are soft pills with tinted backgrounds and high-contrast text.
- Booking calendar selected dates use solid blue; ranges use a light blue tint.
- Filter chips toggle from gray outline to RentFlow blue.
